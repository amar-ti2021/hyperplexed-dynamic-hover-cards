# Dynamic Multi-Card Hover Effect

A modern, multi-card spanning hover effect where a gradient glow follows the user's cursor, lighting up the borders of both the hovered card and its immediate neighbors.

## 🎯 Project Goal
The primary objective of this repository was to reverse-engineer and study an advanced CSS masking technique. 

**Inspiration & Credit:** This project was originally inspired by the excellent breakdown by [Hyperplexed on YouTube](https://youtu.be/htGfnF1zN4g).
